**얼굴로 포인터를 옮기고, 목소리로 터치하는 안드로이드 앱입니다.**

첫 공개 버전입니다. 요리 중이거나, 태블릿을 거치해 두고 보거나, 손을 쓰기 어려운
상황에서 화면을 만지지 않고 조작하기 위해 만들었습니다.

## 📥 설치

아래 **`app-release.apk`** 를 내려받아 설치하세요.

⚠️ **[설치 및 사용 가이드](https://github.com/madcamp-official/26s-w3-c1-01/blob/main/docs/INSTALL.md)를 먼저 읽어 주세요.**
"알 수 없는 출처 허용"과 접근성 권한 설정이 필요해서, 안내 없이 설치하시면 앱이
고장난 것처럼 보입니다.

| | |
| --- | --- |
| 안드로이드 | **8.0 이상** · **전면 카메라 필수** |
| 네트워크 | 음성 인식에 필요 |
| 용량 | 약 55MB |

**아이폰에서는 동작하지 않습니다.**

## 무엇을 할 수 있나요

고개를 돌려 포인터를 옮기고, 말해서 실행합니다.

| 하고 싶은 것 | 말할 것 |
| --- | --- |
| 누르기 | "터치" / "클릭" |
| 뒤로 가기 | "취소" |
| 스크롤 | "위로" / "아래로" (+ "조금", "크게") |
| 좌우 이동 | "다음" / "이전" |
| 끌기 | "잡아" → "놓아" |
| 일시 정지 | "멈춰" → "다시 시작" |
| 잠금 | "잠금" → "해제" |

**쳐다보는 것만으로는 아무것도 실행되지 않습니다.** 화면을 보는 것과 조작하려는
의도를 구분하기 위해 실행은 전부 음성입니다.

## ⚠️ 먼저 알아두실 것

이 앱은 **접근성 서비스 권한**을 요구합니다. 다른 앱 위에 터치를 발생시키려면
안드로이드에서 다른 방법이 없습니다.

**낯선 앱에 이 권한을 주는 것을 망설이는 것이 정상입니다.** 그래서:

- 소스 코드 전체가 이 저장소에 공개되어 있습니다
- **화면 내용을 읽지 않습니다** — 접근성 이벤트 구독을 아예 꺼 두었습니다
- **카메라 영상·얼굴 이미지·음성 녹음을 저장하거나 전송하지 않습니다**
- 진단 데이터 전송은 **기본이 꺼져 있습니다**

## 알려진 한계

- **눈만으로는 조작할 수 없습니다.** 고개 방향이 주 입력이고 눈동자는 미세 조정입니다
- **음성 인식에 인터넷이 필요합니다** (한국어 오프라인 모델이 없는 기기가 많았습니다)
- **삼성 런처 홈 화면에서 좌우 페이지 넘김이 안 됩니다.** 일반 앱에서는 됩니다
- 주변 사람의 발화에 반응할 수 있습니다
- 카메라·마이크를 계속 켜 두기 때문에 발열이 있습니다

## 🙏 피드백 부탁드립니다

**완성품이 아닙니다.** 저희가 가진 기기 몇 대에서만 확인했고, 다른 기기에서 어떻게
동작하는지 모릅니다. **안 되는 것이 있다면 그게 저희에게 가장 필요한 정보입니다.**

앱 안의 피드백 칸에 적어 주시거나,
[이슈](https://github.com/madcamp-official/26s-w3-c1-01/issues)로 남겨 주세요.

이런 내용이 특히 도움이 됩니다:

- **기기 모델명** (예: 갤럭시 S21)
- 어떤 앱에서 무엇을 하려다 안 됐는지
- 인식이 안 된 명령어
- 캘리브레이션이 실패했다면 어느 단계에서인지
