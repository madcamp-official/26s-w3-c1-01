# 프로젝트 계획 — 현재 MVP

## 목표

손을 자유롭게 쓰기 어려운 상황에서 얼굴 방향으로 포인터를 움직이고, 짧은 음성 명령으로 스마트폰의 기본 조작을 실행한다.

## MVP 기능

| 영역 | 내용 |
| --- | --- |
| 포인터 | 전면 카메라 기반 얼굴 방향 추정, 화면 좌표 매핑, 오버레이 표시 |
| 실행 | 터치, 뒤로가기, 상하 스크롤, 좌우 이동 |
| 제어 | 잠금, 잠금 해제, 앱 종료 |
| 보정 | 9점 캘리브레이션과 로컬 프로파일 저장 |
| 안전 | LOCKED 상태에서 수동 해제 버튼 제공 |

## 현재 음성 명령

| 목적 | 명령 |
| --- | --- |
| 탭 | 터치, 클릭, 탭 |
| 뒤로가기 | 취소, 뒤로 |
| 아래 스크롤 | 아래로, 내려, 아래 |
| 위 스크롤 | 위로, 올려, 위 |
| 다음/오른쪽 | 다음, 오른쪽 |
| 이전/왼쪽 | 이전, 왼쪽 |
| 잠금 | 잠금, 끝, 멈춰 |
| 잠금 해제 | 해제 |
| 앱 종료 | 종료 |

## 현재 제외한 기능

- 드래그
- 일시정지/재개
- 스크롤 강도 명령
- 사용자 정의 명령어
- 앱별 전용 최적화

## 검증 관점

1. 실제 기기에서 포인터가 얼굴 움직임을 따라 안정적으로 이동하는가
2. 음성 명령이 한국어 환경에서 충분히 짧은 지연으로 인식되는가
3. 접근성 서비스가 터치/뒤로가기/스크롤을 안정적으로 실행하는가
4. LOCKED 상태에서 음성 또는 수동 버튼으로 빠져나올 수 있는가
5. 카메라/마이크/오버레이/접근성 권한 설명이 사용자에게 충분히 명확한가
